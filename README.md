## ✨ Sections

- ✅ **Hero** — split grid layout, animated outline text hover
- ✅ **Marquee** — scrolling yellow ticker bar
- ✅ **Services** — 6-card grid with hover underline effect
- ✅ **Work** — project showcase with live Unsplash backgrounds
- ✅ **Process** — step-by-step numbered list
- ✅ **Tech Stack** — tag grid with hover
- ✅ **Contact** — split layout with form
- ✅ **Custom cursor** — yellow dot with ring follow
- ✅ **Scroll reveal** — IntersectionObserver animations
- ✅ **Fully responsive** — mobile, tablet, desktop

---

## 🎨 Design

- **Fonts:** Bebas Neue (display) + Epilogue (body)
- **Theme:** Brutalist dark — black background, yellow accent
- **Style:** Editorial agency aesthetic
- **Effects:** Custom cursor, marquee, outline text, hover stripes

---

## 🛠 Tech Stack

| Technology | Usage |

| CSS3 | Layout, animations, grid |
